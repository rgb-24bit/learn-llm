来源占训练集提供什么典型数据集

代码~5-10%逻辑思维、精确GitHub (The Stack, StarCoder)
社交媒体~2%对话风格Reddit, Twitter

二、清洗流水线（5 步）

步骤做什么常用工具
1. HTML 提取去掉 HTML 标签、广告、导航栏jusText, trafilatura, resilience
2. 语言识别判断文档是什么语言FastText, langid, pycld3
3. 启发式过滤按长度/符号比例/重复率筛掉垃圾FineWeb 7 规则, CCNet
4. 去重精确+模糊去重，去掉重复内容MinHash+LSH (datasketch)
5. 模型质量过滤用分类器判断教育/质量FineWeb-Edu classifier, DCLM quality filter

启发式过滤常用规则

规则阈值过滤目标
最小文档长度< 50 字符标题页、404
最大文档长度> 100K 字符数据 dump、日志
标点符号比例> 30%乱码、数据表格
最长行重复比> 30 次模板文本
整句比例< 50%碎片文本
停用词密度< 2%非自然语言
URL 占比> 20%导航页/聚合页

三、去重方法

方法技术复杂度场景
精确去重SHA256 hashO(n)完全相同的文档
模糊去重MinHash + LSHO(n log n)近似重复（抄袭/转载）
语义去重Embedding + 向量相似度高同一事件的多种报道
URL 去重URL 规范化 + 哈希O(n)同一 URL 多次被抓取

七、数据污染类型

类型描述检测难度影响
精确污染测试题原文出现在训练数据易（子串匹配）最高
近似污染小幅改写的测试题中（n-gram 匹配）高
变体污染改写/翻译后的测试题难中
领域污染同领域的非测试内容难低（有帮助）
