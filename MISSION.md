# MISSION: 从零理解大语言模型

## 为什么要学？

主人对 AI 的使用场景、工具实测、效率提升有持续兴趣，日常大量使用 AI 工具。但不想停留在「调用 API」层面——想真正明白底层发生了什么。

就像开车的人懂点发动机原理，能更好地驾驭车一样。

## 学习方式

- **不动手编码**—只聊天，不写代码
- **Fairy 读源码提炼讲解**—Fairy 负责读代码、找资料，用大白话讲给主人听
- **数学仅高考+大学期末水平**—毕业6年没用过，全忘了。避免公式推导，用具体数字算例、表格对比、最土的比喻
- **从零开始，逐个概念堆叠**—不跳跃，每步打实了再往前走
- **每轮 Q&A 存结构化笔记**—笔记 git commit 到仓库
- **学习计划不设时间，只分阶段**—学多少算多少
